Installation

Guide

The installation of University pages is straightforward, the database is automatically built and some important data is included on the first api call. If your database configuration is not the default it can be changed at /inc/DB/db.php.

Browser

This site may function with other browsers however due to use of ES6 features and a unique drag and drop feature, I have only found the FireFox Nightly to perform completely.

oAuth

The only major issue with installation is a problem with the oAuth login system. This system uses google authentication to authorise users, as such it requires 3rd party configuration to function correctly. UniPages by default will function with oAuth ONLY if it is hosted at localhost:8080 this is due to google denying any authorisation that redirects to a public IP address and requiring you to pre authorise redirect URl's.

I have included alternative methods of login for testing purposes but to use oAuth you must host at localhost:8080
